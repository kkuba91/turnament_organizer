Turnament Organizer

Application for organizing chess tournaments with pairing systems.

T(o)urnament Organizer (polonisation of "tournament"). It is for all kinds of use (non-professional, commercial, private, etc.). What is important it is for fun.

Quick Start
  1. Click Open to select an existing or create a new tournament
  2. Add players to the start list
  3. Choose pairing system and number of rounds
  4. Click Start to begin the tournament
  5. Enter results for each round
Information
Pairing systems: Swiss, Round-robin, Single elimination
Results: +1 ½ +0
Features: byes, player suspension, ELO statistics
Data: automatic save to SQLite database
